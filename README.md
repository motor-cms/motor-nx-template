# motor-nx-template

## Installation

    mkdir packages
    cd packages
    git clone git@github.com:motor-cms/motor-nx-core.git
    git clone git@github.com:motor-cms/motor-nx-media.git
    git clone git@github.com:motor-cms/motor-nx-backend.git
    cd ..
    // IMPORTANT: use nodejs 18.x
    npm install
    npm run serve
